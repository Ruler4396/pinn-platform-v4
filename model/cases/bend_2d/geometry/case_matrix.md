# bend_2d case matrix

## 1. 几何定义

- 参考宽度：`W = 200 μm`
- 入口直段：`L_in = 4W`
- 出口直段：`L_out = 6W`
- 弯道角度：`θ = 90°`
- 曲率半径：`R_c ∈ {6W, 5W, 4W, 3.5W, 3W}`

## 2. case 列表

| case_id | R_c/W | θ | 用途 | 备注 |
|---|---:|---:|---|---|
| B-base | 6.0 | 90° | 单工况基线 | 首个必须跑通的 case |
| B-train-1 | 6.0 | 90° | 训练 | 低曲率 |
| B-train-2 | 5.0 | 90° | 训练 | 低-中曲率 |
| B-train-3 | 4.0 | 90° | 训练 | 中曲率 |
| B-val | 3.5 | 90° | 验证 | 中-高曲率 |
| B-test-1 | 3.0 | 90° | 测试 | 高曲率挑战 |
| B-test-2 | 4.0 | 60° | 测试 | 转角泛化扩展 |

## 3. 关键观测截面

建议固定以下截面用于曲线对比与物理量统计：

- `S1`：入口直段中部
- `S2`：弯前截面（入弯前 `0.5W`）
- `S3`：弯中截面（45° 中截面）
- `S4`：弯后截面（出弯后 `0.5W`）
- `S5`：出口直段中部

## 4. 重点评价指标

- 弯前/弯中/弯后截面速度分布误差
- 内弯 / 外弯壁面邻域误差
- 总压降误差
- 壁面 no-slip 最大误差
- 散度均值 / 最大值
