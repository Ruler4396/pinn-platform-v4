# model/results

这个目录保留给整合仓的模型结果与论文展示资产使用。

当前仓库已经纳入一批用于完整项目展示的 `v4` 结果快照，包括：

- `pinn/` 下的收缩流道正式主线与稀疏/噪声对照 run
- 对应训练日志、评估日志与 `best.ckpt`
- `field_map_checks/` 下的误差导图检查与章节素材
- `thesis_assets/` 下的第五章图表与清单文件

这些内容主要用于：

- 开源展示完整研究链路
- 保留论文图表来源
- 让整合仓在离线状态下也能看到关键 run 的配置、日志、指标和素材

结果目录中的路径字段已经尽量改为仓库内相对口径。少量 `historical/pinn_v3/...` 仅表示这些图表来源于更早阶段的历史结果仓，而不是当前整合仓内的实际路径。

如果后续继续产生大体量实验产物，建议仍按以下原则控制入库范围：

- 优先保留 `config.json`、`metrics.json`、`history.csv`、汇总 `csv/json`
- 只为正式展示 run 保留必要的日志、checkpoint 和图表
- 其余中间产物优先放本地或外部归档，避免主仓库历史过重
